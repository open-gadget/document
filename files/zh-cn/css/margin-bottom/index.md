## 介绍

设置与元素相关联的盒子模型的下外边距，这个值可以为负值。

## 语法

```css
/* <length> values */
margin-bottom: 0.5em;
margin-bottom: 0;
margin-bottom: 2px;

/* <percentage> value */
margin-bottom: 10%;
```

### 取值

-   `<length>`

以固定值为外边距。

-   `<percentage>`

相对于包含块的宽度，以百分比值为外边距。

### 标准化语法

```css
margin-bottom: <length> | <percentage>
```
