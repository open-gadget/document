## 介绍

设置与元素相关联的盒子模型的左外边距，这个值可以为负值。

## 语法

```css
/* <length> values */
margin-left: 0.5em;
margin-left: 0;
margin-left: 2px;

/* <percentage> value */
margin-left: 10%;
```

### 取值

-   `<length>`

以固定值为外边距。

-   `<percentage>`

相对于包含块的宽度，以百分比值为外边距。

### 标准化语法

```css
margin-left: <length> | <percentage>
```
